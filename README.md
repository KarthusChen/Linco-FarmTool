# Linco-FarmTool · 收菜闹钟桌宠

老婆沉迷王者农场，做个小工具给老婆用。

一只蹲在桌面角落、抱着菜苗的花枝鼠 🐭，帮你盯着王者农场地里的**浇水 / 收菜时间**，到点就跳起来喊脑婆上号。桌宠风格——常驻桌面、始终可见、几乎不占资源。

专为"种下作物 → 按最优节奏浇水 → 准点收割"这套流程打造：选好作物一键排好整串闹钟，再也不用自己掐表。

## ✨ 功能

### 🐭 桌宠
- 透明无边框、始终置顶、可拖动的小老鼠，默认待在屏幕右下角
- 常驻系统托盘，右键可快捷操作
- 高分屏 DPI 感知，不发虚

### 🌱 一键种菜排闹钟（核心）
点「种菜」按最优浇水节奏自动排好整串闹钟，两种模式：
- **现在开种** —— 刚种下就用，自动排 3 个闹钟（第 2、3 次浇水 + 浇水秒熟收割）
- **种了有一会了** —— 照着游戏里的**剩余成熟时间**和**水分剩余时间**填，自动算出从现在起该几点浇水、几点收割
- 内置全部 **56 种作物**（1 / 8 / 16 / 32 小时四档），选作物自动带出周期
- 实时预览每步时刻，填错（超过上限等）当场标红提示

### ⏰ 闹钟
- 多个闹钟并存，实时倒计时
- 也能手动加单个闹钟：支持**倒计时**（X 小时 Y 分钟后）或**指定时间**（今天/明天几点）
- 本地持久化，关掉重开不丢

### 🔔 到点提醒（登录提前量 + 无声）
- **提前提醒**：默认提前 3 分钟就响，给你留出打开游戏登录的时间；托盘可切换 准点 / 1 / 3 / 5 / 10 分钟
- 到点后小老鼠**一边跳一边喊「脑婆脑婆上号啦！」**，直到你点一下它才停下闭嘴
- 同时弹 Windows 系统通知、窗口闪烁提醒
- 到点那条闹钟**整行变红**，满 **10 分钟自动删除**

## 🧮 计算原理（肝帝模式）

浇水排程严格按游戏机制推导（`T` = 作物自然周期）：

| 规则 | 公式 |
|---|---|
| 湿润持续时间 | `W = T / 3` |
| 浇水减时 | `减时 = (W − w) / 4`（`w` = 当前湿润剩余） |
| 可浇水阈值 | `W − w ≥ T / 30`（约 10% 蒸发后才能浇） |
| 最短总耗时 | `11T / 15` |

**最优策略**：干透就浇（每次减 `T/12`），末次在还湿润时提前 `4R/5` 浇水秒熟。
由「刚种下」`(R=T, w=0)` 出发可精确复现浇水时刻 `0 / T3 / 2T3 / 11T15`，与游戏完全一致（如 16h 作物 → 11 小时 44 分成熟）。

## 🛠 技术栈

- **C# + WPF (.NET 8)** — Windows 原生，透明置顶窗口、低资源占用
- **System.Text.Json** — 闹钟与设置本地持久化（`%AppData%\LincoFarmTool\`）
- **WinForms NotifyIcon** — 系统托盘 + 通知气泡
- **Win32 FlashWindowEx** — 到点窗口闪烁提醒

小老鼠、界面全部用 WPF 矢量图形 + 样式手绘，无外部图片/音频资源，打包不丢件。

## 🚀 运行

```sh
cd src/LincoFarmTool
dotnet run
```

启动后小老鼠出现在屏幕右下角。想快速看到点效果：点 `＋` 添加一个「倒计时 0 小时 1 分钟」的闹钟——因默认提前 3 分钟，会立即触发。

## 📦 打包成 exe（分发给非技术用户）

打出来的是**单文件、自包含的绿色 exe**（约 68 MB，内置 .NET 运行时），双击即用、无需安装环境。

**本地一键打包**：
```powershell
./publish.ps1        # 产物在 dist/LincoFarmTool.exe
```

**自动发布（推荐）**：打一个版本 tag，GitHub Actions 会自动构建并发布到 Releases：
```sh
git tag v1.0.0
git push origin v1.0.0
```
> ⚠️ 要让别人能下载、以及应用内「检查更新」能用，**仓库需设为 public**（私有仓库的 Release 资源无法匿名下载）。

### 安装 / 更新 / 卸载

| 操作 | 怎么做 |
|---|---|
| **安装** | 下载 `LincoFarmTool.exe`，放到固定文件夹双击运行；托盘勾选「**开机自启**」即可开机自动启动 |
| **更新** | 托盘「**检查更新**」——有新版一键下载、自动替换重启，数据不丢（启动时也会静默检查一次） |
| **卸载** | 托盘「**彻底卸载 / 清理**」清除开机自启项与本地数据并退出，再删掉 exe 即可 |

> 数据（闹钟、设置）存在 `%AppData%\LincoFarmTool\`，与 exe 分离，更新不受影响。

    ├── Models/
